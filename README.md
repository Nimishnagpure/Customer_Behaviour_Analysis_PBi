The objective of this project is to analyze customer shopping behavior, uncover valuable business insights, and develop an interactive dashboard that supports data-driven decision-making. It demonstrates practical skills in data cleaning, feature engineering, SQL analysis, business intelligence, and dashboard development, making it a strong portfolio project for aspiring Data Analysts.
